## Boilerplate Flask web app

For practice purposes. But once finished, it can be easily copied and adapted to suit whatever needs.

### Tech Stack
- Web: Flask
- ORM: SQLAlchemy
- Database: PostgreSQL
- Containerization: Docker
- WSGI Server: Gunicorn
